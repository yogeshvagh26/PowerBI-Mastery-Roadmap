# Power Bi - Learning Roadmap

### Phase 1: Power BI Fundamentals

* Introduction to Power BI
* Business Intelligence Concepts
* Installing and Setting Up Power BI Desktop
* Understanding the Power BI Interface
* Data Sources and Data Connections
* Import Mode vs DirectQuery
* Understanding Power BI Workflow

    ----

### Phase 2: Data Preparation with Power Query

* Introduction to Power Query Editor
* Importing Data from Excel, CSV, Databases, and Web Sources
* Data Cleaning Techniques
* Handling Missing and Duplicate Data
* Data Transformation
* Merging Queries
* Appending Queries
* Creating Custom Columns
* Data Profiling and Data Quality

    ----

### Phase 3: Data Modeling

* Understanding Data Models
* Tables, Columns, and Relationships
* One-to-One, One-to-Many, and Many-to-Many Relationships
* Star Schema and Snowflake Schema
* Primary and Foreign Keys
* Relationship Management
* Calculated Columns vs Measures

    ----

### Phase 4: DAX (Data Analysis Expressions)

* Introduction to DAX
* DAX Syntax and Fundamentals
* Aggregation Functions
* Logical Functions
* Text Functions
* Date and Time Functions
* Filter Context and Row Context
* CALCULATE Function
* Time Intelligence Functions
* Advanced DAX Techniques

    ----

### Phase 5: Data Visualization

* Understanding Visualizations
* Creating Charts and Graphs
* Tables and Matrix Visuals
* Cards and KPIs
* Maps and Geographic Analysis
* Slicers and Filters
* Drill Down and Drill Through
* Conditional Formatting
* Custom Visuals

    ----

### Phase 6: Dashboard and Report Design

* Designing Professional Reports
* Choosing the Right Visuals
* Dashboard Layout and Formatting
* Themes and Templates
* Navigation and Bookmarks
* Tooltips
* User Experience Best Practices

    ----

### Phase 7: Advanced Power BI

* Advanced Data Modeling
* Advanced DAX Calculations
* Parameters and Dynamic Reports
* Row-Level Security (RLS)
* Performance Optimization
* Query Optimization
* Incremental Refresh
* Composite Models

    ----

### Phase 8: Power BI Service and Collaboration

* Introduction to Power BI Service
* Publishing Reports
* Creating and Sharing Dashboards
* Workspaces
* Apps
* Data Refresh Scheduling
* Gateways
* Collaboration and Permissions

    ----

### Phase 9: Power BI Administration and Security

* Workspace Management
* Data Security Concepts
* Row-Level Security Management
* Governance and Best Practices
* Monitoring and Performance Management

    ----

### Phase 10: Real-World Projects

Build practical projects to apply your knowledge, such as:

* Sales Performance Dashboard
* Financial Analysis Dashboard
* HR Analytics Dashboard
* Inventory Management Dashboard
* Customer Behavior Analysis Dashboard
* Marketing Performance Dashboard
* Executive KPI Dashboard
